# Changelog

All notable changes to Fikra will be documented in this file.

## [0.2.0] - 2026-02-22

### Added
- Initial public release
- GGUF Q4_K_M quantized model
- Automatic model download from Hugging Face
- `reason()` method for inference
- Fine-tuned on GSM8K (math) and Dolly 15k (instructions)

### Technical
- Base: Falcon-E-1B-Base
- Size: ~700MB
- Context: 2048 tokens
- License: Apache 2.0

## [0.1.0] - 2026-01-XX

### Added
- Private alpha release
- Basic inference capabilities
